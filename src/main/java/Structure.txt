Напоминание:
	Сотрудник может изменять статус только до выполнено, 
	а руководитель до завершено. После изменения статуса на завершено,
	сотрудник не может изменять статус и отчет.
	
	Просмотр задач с сортировкой и выборкой
	Добавление редактора ролей
	

Роли:
	Сотрудник
	Директор
	Руководитель
	

Действия:
Авторизация
	Сотрудника:
		Написать отчет
		Просмотр отчетов
		
		Просмотр текущих задач
		Изменение статуса выполнения задачи
		Добавление отчета к задаче
		Редактирование отчета к задаче
	
	Руководителя и директора:
		Написать отчет
		Просмотр отчетов
		
		Просмотр задач
		Создание задачи
		Редактирование задачи
		Удаление задачи
		Перевод задачи в архив
	
	
Сущности:
	Пользователь
	Задача
		Отчет по задаче
	Отчет
	
	
Данные:
	Пользователь
		id, ФИО, текстовое поле, соль, хешСоль, роли, статус работы
	Задача	
		id,  название, текстовое поле, создатель, работник, список тегов, дата/время создания, дата/время дедлайна,
		дата/время завершения, статус, (таблица доступа)
			Отчет	
				() 
	Отчет	
		id,  текстовое поле, список тегов, отправитель, список получателей, дата/время создания,
		список дат и времени изменений (таблица доступа)
		
Функции:
Аутентификация
	Пользователь
		Ведение лога
		
		Просмотр отчетов
		Написание отчета
		Просмотр задач
		Создание задачи
		Редактирование задачи
		Удаление задачи
		Перевод задачи в архив
